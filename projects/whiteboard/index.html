Whiteboard

Software to scan the contents of a whiteboard.

Made for the We Can Code Hackathon

How it works:

The software needs to be connected to a Wiimote (Nintendo Wii controller) via bluetooth, since this type of controller has an infrared receiver at the tip. The Wiimote is placed pointing to the whiteboard and, for the markers to be detected, they have an infrared LED on the back. The software receives the movements of the markers and, from these, generates a highly similar image to the one on the board.
